Before you run the tracker
Make sure that you fulfill all the requirements: Python 3.8 or later with all requirements.txt dependencies installed, including torch>=1.7. To install, run:

	pip install -r requirements.txt

# for yolo+deepsort common commands

# --source
#  video path 
# /home/natsu/AIST/video/2021-07-06-17-30-01_Tsukuba.mp4  
# if want to using webcam 
# --source 0 

# --yolo_weights 
# yolov5 model path  default yolov5x.pt

# --img-size 640 or 1280 
# The size of the feature map of the image can be specified when the YOLO model is processed (must be a multiple of 32)

# --conf-thres 
# Threshold for the presence of an object in each anchor

# --save-txt 
# Store the location of the tracked object in each frame

# --save-vid
# save video result after yolo+deepsort

# --classes 
# yolo specifies the category of the detected object (total 80 classes)
# 0 2 mean person and car ã€€

# --show-vid
# You can dynamically check the results

# example run original code (yolo+deepsort_original folder)

	python track.py --source /home/natsu/AIST/video/2021-07-06-17-30-01_Tsukuba.mp4 --yolo_weights /home/natsu/Yolov5_DeepSort_Pytorch/yolov5/weights/yolov5x.pt --img-size 1280 --conf-thres 0.1 --save-txt --classes 0 2 --save-vid --show-vid

# example run finetune code (yolo+deepsort_finetune folder)
	python new_track.py --source /home/natsu/AIST/video/2021-07-06-17-30-01_Tsukuba.mp4 --yolo_weights /home/natsu/Yolov5_DeepSort_Pytorch/yolov5/weights/yolov5x.pt --img-size 1280 --conf-thres 0.1 --save-txt --classes 0 2 --save-vid --show-vid
